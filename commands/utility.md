## Voting

### Voting Add

{% hint style="success" %}
@Manager\#9545 **voting add \[params\]**
{% endhint %}

{% hint style="info" %}
Creates a new voting
{% endhint %}

```text
@Manager#9545 voting add Title;Description
```

### Voting Close

{% hint style="success" %}
@Manager\#9545 **voting close \[id\]**
{% endhint %}

{% hint style="info" %}
Closes the voting
{% endhint %}

```text
@Manager#9545 voting close 1
```
